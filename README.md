

💰 Zakat Calculator (C++ Console Program)

This is a simple C++ console application that calculates Zakat (2.5%) based on the user’s religion and total balance.

📌 Features

Prompts the user to confirm if they are Muslim (y/n)

Takes total balance as input

Calculates Zakat at 2.5% if:

User is Muslim

Balance is 500,000 or more

Displays clear messages if Zakat is not applicable

🛠️ Technologies Used

Language: C++

Compiler: GCC / MinGW / MSVC (any standard C++ compiler).
